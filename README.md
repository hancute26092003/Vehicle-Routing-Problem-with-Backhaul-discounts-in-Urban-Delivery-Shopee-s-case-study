# Vehicle-Routing-Problem-with-Backhaul-discounts-in-Urban-Delivery-Shopee-s-case-study
Addresses inefficiencies in Shopee’s urban delivery network in Ho Chi Minh 
City by leveraging third-party logistics (3PL) services offering backhaul discounts, a 
comprehensive vehicle routing problem (VRP) model is developed to optimize pick-up 
and delivery operations. Using real-world data from Shopee’s Gateway Hub and Local 
Hubs, the model incorporates constraints such as time windows, a heterogeneous fleet, 
and docking capacity. A Branch-and-Price algorithm is applied to minimize 
transportation costs through backhaul integration while improving task coordination.
